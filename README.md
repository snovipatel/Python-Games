# Python-Games

# Hangman Game

A Python console game where players guess a random word one letter at a time.  
Players have up to seven wrong guesses before losing.

## Features
- Randomly selects a word for each game
- Tracks wrong guesses and remaining attempts
- Displays progress with blanks and correct letters
- Simple console interface

# Rock Crusher Game

A Python game where a bouncing ball hits user-placed rocks.  
Rocks change colors based on hit count, and the ball reacts with random speed and color changes.

## Features
- Interactive gameplay with user-placed obstacles
- Ball changes color and speed on hitting rocks
- Dynamic rock behavior based on hits
- Simple graphical interface
